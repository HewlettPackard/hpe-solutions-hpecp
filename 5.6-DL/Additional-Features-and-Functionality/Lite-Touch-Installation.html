# Lite Touch Installation (LTI)

To enhance the experience of deploying HPE Ezmeral Runtime Enterprise 5.6.0 using automation scripts even further, the Lite Touch Installation (LTI) enable administrator to deploy HPE Ezmeral Runtime Enterprise 5.6.0 using fewer playbooks. The prerequisites for the LTI deployment method are availability of iLO access to the servers and the servers are connected to the application network and have the required storage available to deploying HPE Ezmeral Runtime Enterprise 5.6.0 successfully along with Kubernetes cluster running HPE Ezmeral Data Fabric.

The high-level flow for the LTI deployment method is as follows:

Figure 61. Lite Touch Installation High Level Flow

The LTI enables end-end deployment of HPE Ezmeral Runtime Enterprise 5.6.0 starting with the Operating System deployment on compute to installing k8s based HPE Data Fabric storage and registering it with the Platform for k8s multi compute cluster usage. For more information on HPE Ezmeral Runtime Enterprise 5.6.0 Lite Touch Installation (LTI) method and scripts refer: https://github.com/hewlettpackard/hpe-solutions-hpecp (opens new window)
