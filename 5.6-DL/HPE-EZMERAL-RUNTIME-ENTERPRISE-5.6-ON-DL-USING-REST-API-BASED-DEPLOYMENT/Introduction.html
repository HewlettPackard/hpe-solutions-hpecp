# Introduction

This document describes the steps required to deploy the HPE Ezmeral Runtime Enterprise 5.6.0 environment running Suse Linux Enterprise Server 15 SP3 on HPE ProLiant DL325 & DL385 Gen11 Servers. It is intended to be used in conjunction with Ansible playbooks and Python scripts found at https://github.com/hewlettpackard/hpe-solutions-hpecp (opens new window). This document should be reviewed in its entirely and the installation user should understand all prerequisites and procedure prior to installation. It is also recommended that the installation user reviews the HPE Ezmeral Runtime Enterprise installation process as described in https://docs.containerplatform.hpe.com/56/ (opens new window)
