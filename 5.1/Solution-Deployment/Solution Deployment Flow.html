# Standards used in this document

This document makes use of the following standard terms:

  1. Installation user or installer - Individual or individuals responsible for carrying out the installation tasks to produce a functional HPE Ezmeral Container Platform solution on HPE Synergy and HPE Apollo 4200 Servers.

  2. Installer machine or installer VM - The system that is capable of connecting to various components within the solution and is used to run most of the key commands. In this solution, this machine also serves as the Ansible Engine host. For more information, refer to the section Installer machine in this document.

NOTE

The scripts and reference files provided with this document are included as examples on how to build the solution and they are not supported by Hewlett Packard Enterprise. It is expected that the scripts and reference files will be modified to meet the requirements of the deployment environment by the user prior to installation.
